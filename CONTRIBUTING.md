# Contributing

Thanks for your interest in Declaw!

## This repository is a mirror

Development happens in a private monorepo; this repository receives
snapshot updates from it. In practice:

- **Issues — yes, please.** Bug reports and feature requests opened here
  are read and triaged.
- **Pull requests — welcome, with a caveat.** We can't merge PRs directly
  (the next sync would overwrite them). Accepted changes are ported into
  the monorepo with credit to you in the CHANGELOG, and land here with
  the next sync.
- Git history here is intentionally coarse. `CHANGELOG.md` is the
  detailed, authoritative record of changes.

## Security issues

Please do not open public issues for suspected vulnerabilities. Use
GitHub's private vulnerability reporting on this repository
(**Security → Report a vulnerability**).
